Grid Terminology

Grid Container The parent element where display: grid is contained
Grid Item The child elements of which the parent element has grid applied
Grid Line The vertical and horizontal lines that make up the structure of the grid
Grid Track The area that defines the columns and rows of a grid

Grid Terminology

Grid Cell A single unit of the grid
Grid Area The total space between four grid lines
